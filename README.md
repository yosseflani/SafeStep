# SafeStep

## אודות הפרויקט

SafeStep היא אפליקציית מובייל שפותחה באמצעות Flutter ומיועדת לעיוורים ולקויי ראייה.  
האפליקציה מזהה מכשולים בזמן אמת באמצעות מצלמת הטלפון ומתריעה למשתמש באמצעות:

- התראות קוליות
- רטט
- צפצופי אזהרה
- פקודות קוליות אופליין

המטרה המרכזית של הפרויקט היא לסייע בהתמצאות בטוחה יותר בסביבה יומיומית באמצעות טכנולוגיות בינה מלאכותית וראייה ממוחשבת.

---

# תכונות מרכזיות

## זיהוי מכשולים בזמן אמת
האפליקציה משתמשת במצלמת המכשיר ובמודל YOLOv8 לזיהוי אובייקטים שונים בסביבה.

האובייקטים הנתמכים כוללים:

- אנשים
- רכבים
- אופניים
- אוטובוסים
- משאיות
- רמזורים
- תמרורים
- בעלי חיים
- ריהוט ומכשולים נוספים

---

## מערכת חישוב סיכון חכמה

לכל אובייקט מחושב ציון סיכון בין 0 ל־100 לפי:

- סוג האובייקט
- גודל האובייקט בתמונה
- מיקום האובייקט במסך
- האם האובייקט מתקרב למשתמש
- קצב שינוי הגודל
- תנועת האובייקט

---

## מערכת התראות מתקדמת

### רטט
הטלפון מרטיט בהתאם לסוג ורמת הסיכון של האובייקט.

### התראות קוליות
האפליקציה מקריאה את שם האובייקט ורמת הסכנה.

לדוגמה:
- "מכונית קרוב מאוד"
- "אדם לפניך"

### צפצוף חירום
כאשר מזוהה סכנה גבוהה במיוחד מופעל גם צפצוף אזהרה.

---

## פקודות קוליות אופליין

האפליקציה תומכת בזיהוי פקודות קוליות בעברית ללא חיבור לאינטרנט באמצעות Whisper ו־sherpa-onnx.

### דוגמאות לפקודות:
- "הפעל זיהוי"
- "עצור זיהוי"
- "פתח הגדרות"
- "הפעל רטט"
- "כבה רטט"

---

## מסך הגדרות

האפליקציה כוללת מסך הגדרות מתקדם המאפשר:

- בחירת שפה
- בחירת קול
- שינוי מהירות דיבור
- הפעלה/כיבוי רטט
- בדיקת קול
- איפוס להגדרות ברירת מחדל

# טכנולוגיות וכלים

## Flutter & Dart
פיתוח האפליקציה בוצע באמצעות Flutter.

## YOLOv8
מודל לזיהוי אובייקטים בזמן אמת.

## TensorFlow Lite
הרצת מודל AI ישירות על המכשיר.

## sherpa-onnx + Whisper
זיהוי דיבור אופליין בעברית.

## Text To Speech
הקראת התראות קוליות למשתמש.

## Sensors
שימוש באקסלרומטר לזיהוי תנועה של המשתמש.

# רמות התראה

| רמת סיכון | פעולה             |
| --------- | ----------------- |
| 0–30      | ללא התראה         |
| 30–40     | רטט בלבד          |
| 40–65     | רטט + קול         |
| 65+       | רטט + צפצוף + קול |
